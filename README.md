# Battle of the Bandwidth

A modern web application for tracking and monitoring network bandwidth usage.

## Features

- Bandwidth tracked over time
- Bandwidth performance charted to visualize performance issues
- React web interface
- Go API server
- Containerized deployment support

## Project Structure

- [`frontend/`](frontend/README.md) - React-based web interface
- `backend/` - Server-side application
- `docker-compose.yml` - Docker Compose configuration for easy deployment

## Installation

### Using Docker (Recommended)

```bash
# Clone the repository
git clone https://github.com/yourusername/battle-of-the-bandwidth.git
cd battle-of-the-bandwidth

# Start the application using Docker Compose
docker-compose up
```
